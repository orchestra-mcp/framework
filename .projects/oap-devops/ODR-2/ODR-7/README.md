# ODR-7: Code Signing & Notarization

**Type**: Story | **Status**: done | **Points**: -

As a user, I want signed applications so that I can trust the software and avoid security warnings

## Acceptance Criteria

- [x] macOS apps signed and notarized
- [x] Windows apps signed with Authenticode
- [x] No security warnings on first launch
- [x] Certificates stored securely in GitHub Secrets
