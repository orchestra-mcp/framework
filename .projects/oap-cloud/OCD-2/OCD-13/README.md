# OCD-13: Login, Register & Password Reset (Fortify)

**Type**: Story | **Status**: backlog | **Points**: 5

As a user, I want to register, log in, and reset my password so that I can securely access the platform.

## Acceptance Criteria

- [ ] Login page with email + password works via Fortify
- [ ] Registration page with name + email + password works via Fortify
- [ ] Password reset via email link works via Fortify
- [ ] All forms validate inputs and show errors
- [ ] Successful login redirects to dashboard
- [ ] All pages are Inertia React components with Tailwind styling
