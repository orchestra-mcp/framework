# OCD-23: Documentation Search

**Type**: Story | **Status**: backlog | **Points**: 3

As a developer, I want to search within documentation so that I can find specific topics or references without manually browsing.

## Acceptance Criteria

- [ ] Full-text search across all documentation
- [ ] Search results show title, path, and content snippet
- [ ] Search is fast (indexed or pre-built)
- [ ] Search input accessible from sidebar
- [ ] Results link directly to matching doc page
