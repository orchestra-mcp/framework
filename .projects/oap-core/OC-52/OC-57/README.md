# OC-57: Register default integrations (Cloudflare, GitHub, Google)

**Type**: Story | **Status**: backlog | **Points**: 5

As a user, I want pre-configured integrations for popular services so that I can quickly connect my accounts without manual setup

## Acceptance Criteria

- [ ] Cloudflare integration with API token validation
- [ ] GitHub OAuth integration with repo and user scopes
- [ ] Google OAuth integration with email and profile scopes
- [ ] Each integration includes validation endpoint
- [ ] Integration fields map to .env variables where needed
- [ ] Help text and placeholders guide users
