# OCD-10: Install & Configure Auth Packages (Jetstream, Fortify, Sanctum)

**Type**: Story | **Status**: backlog | **Points**: 5

As a developer, I want Jetstream with Teams, Fortify, and Sanctum installed and configured so that the authentication foundation is ready for the auth system epic.

## Acceptance Criteria

- [ ] Jetstream installed with Inertia stack and Teams support

- [ ] Sanctum installed and configured for API token auth
- [ ] Jetstream published views work with Inertia + React
- [ ] Default auth pages (login, register) render correctly
