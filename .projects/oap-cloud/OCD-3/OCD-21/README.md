# OCD-21: Web Marketplace

**Type**: Story | **Status**: backlog | **Points**: 8

As a visitor, I want to browse available extensions in a web marketplace and install them to my desktop app so that I can discover and add capabilities to my IDE.

## Acceptance Criteria

- [ ] Extension catalog browsable by category
- [ ] Search and filter extensions
- [ ] Extension detail page with README, screenshots, changelog
- [ ] Install button triggers deep link orchestra://install/{extension-id}
- [ ] Free vs paid badge
- [ ] Rating/reviews display
- [ ] Responsive grid layout
