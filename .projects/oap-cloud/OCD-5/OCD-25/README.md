# OCD-25: Synced Tasks (Project/Epic/Story/Task Hierarchy)

**Type**: Story | **Status**: backlog | **Points**: 8

As an employee, I want to view and manage my tasks in a project/epic/story/task hierarchy synced from the IDE so that I can track my work in one place.

## Acceptance Criteria

- [ ] Task hierarchy: Project -> Epic -> Story -> Task
- [ ] Full markdown rendering for task descriptions
- [ ] Status workflow: backlog -> in_progress -> review -> done
- [ ] Filter by status, project, priority
- [ ] Sort by priority, due date, status
- [ ] Search across tasks
- [ ] Real-time sync from IDE via WebSocket
