# OCD-14: OTP / 2FA & Magic Login

**Type**: Story | **Status**: backlog | **Points**: 5

As a security-conscious user, I want to enable two-factor authentication with recovery codes and optionally log in via magic email links so that my account is more secure.

## Acceptance Criteria

- [ ] 2FA setup page shows QR code for authenticator app
- [ ] 2FA verification works during login
- [ ] Recovery codes are generated and displayed on setup
- [ ] Recovery codes can regenerate
- [ ] Magic login sends email with one-time link
- [ ] Magic login link authenticates user and redirects to dashboard
- [ ] Magic login links expire after 15 minutes
