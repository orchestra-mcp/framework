# OCD-30: Team Management (Invite, Roles, Assignments)

**Type**: Story | **Status**: backlog | **Points**: 5

As a team owner, I want to invite/remove members, assign roles, and set task assignments so that I can manage my team effectively.

## Acceptance Criteria

- [ ] Invite members by email with role selection
- [ ] Remove members from team
- [ ] Change member roles
- [ ] Assign tasks to team members
- [ ] View member profiles and activity summaries
- [ ] Bulk invite via CSV upload
- [ ] Pending invitations list with resend/cancel
