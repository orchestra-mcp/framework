# ODS-43: Chrome extension adapter

**Type**: Story | **Status**: backlog | **Points**: 5

As a developer, I want a Chrome extension adapter so that the design system works in Chrome extension contexts with proper CSP compliance.

## Acceptance Criteria

- [ ] Content Security Policy compliant styling
- [ ] No inline styles or scripts
- [ ] Proper shadow DOM support
- [ ] Extension-specific optimizations
- [ ] Message passing for theme sync
- [ ] Documentation for extension usage
