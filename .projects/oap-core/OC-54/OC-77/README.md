# OC-77: Create FeatureFlagService for dynamic feature control

**Type**: Story | **Status**: backlog | **Points**: 5

As an admin, I want to enable/disable IDE features remotely so that I can control feature rollout and manage beta features

## Acceptance Criteria

- [ ] Service checks feature flags from server
- [ ] Flags cached locally
- [ ] Real-time updates via WebSocket
- [ ] Supports global, user, workspace scopes
- [ ] Events fired on flag changes
