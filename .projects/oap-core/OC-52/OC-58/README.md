# OC-58: Create account management UI

**Type**: Story | **Status**: backlog | **Points**: 5

As a user, I want a UI to add, test, and manage my service accounts so that I can easily configure integrations

## Acceptance Criteria

- [ ] Settings panel lists all available integrations
- [ ] Add account form with dynamic fields based on integration
- [ ] Test connection button validates credentials
- [ ] Account list shows status (active/error) and last validated
- [ ] Edit and delete operations work correctly
- [ ] Uses design system components
