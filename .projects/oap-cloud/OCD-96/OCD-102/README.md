# OCD-102: Admin Subscription Management

**Type**: Story | **Status**: backlog | **Points**: 8

As an admin, I need a dashboard to manage subscriptions so that I can manually activate, extend, or cancel user subscriptions.

## Acceptance Criteria

- [ ] Admin can view all subscriptions
- [ ] Admin can create manual subscriptions
- [ ] Admin can extend subscriptions
- [ ] Admin can cancel subscriptions
- [ ] Admin can create custom subscriptions
- [ ] Real-time updates work
