# OCR-2: Dynamic Tabs System

**Type**: Epic | **Status**: backlog | **Priority**: high

Build a full tab management system at src/app/Chrome/Tabs/ with openTab/closeTab/focusTab APIs. Supports terminal tabs, diff tabs, editor tabs, and custom extension tabs. Features: drag-and-drop reordering, close buttons, overflow handling (scroll/dropdown), tab context menus, active tab state persistence, and split view support (horizontal/vertical). Target directory: src/app/Chrome/Tabs/
