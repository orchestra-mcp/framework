# OCD-29: Team Activity Widgets & Community Page

**Type**: Story | **Status**: backlog | **Points**: 5

As a team owner, I want to see team-wide activity widgets and a community page showing what each member is working on so that I can monitor team productivity in real time.

## Acceptance Criteria

- [ ] Team members online/offline indicators
- [ ] Tasks completed by team (today/week) widget
- [ ] Total time tracked by team widget
- [ ] Active sprints overview widget
- [ ] Community page listing all team members
- [ ] Real-time activity indicator per member (current task, last commit, timer status)
- [ ] All updated via WebSocket presence channels
