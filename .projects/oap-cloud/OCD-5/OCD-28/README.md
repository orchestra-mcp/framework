# OCD-28: Notifications & Sprint Tracking

**Type**: Story | **Status**: backlog | **Points**: 5

As an employee, I want an in-app notification center and sprint tracking with external tool sync so that I stay informed and aligned with my team's sprints.

## Acceptance Criteria

- [ ] In-app notification center with mark read/unread
- [ ] Filter notifications by type
- [ ] Agent alerts from MCP displayed
- [ ] Sprint view synced from Jira/Linear/GitHub
- [ ] Current active sprint display
- [ ] My tasks in sprint view
- [ ] Sprint burndown chart
