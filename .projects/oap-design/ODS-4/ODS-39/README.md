# ODS-39: Modal system (Modal, Dialog, Drawer, Sheet)

**Type**: Story | **Status**: backlog | **Points**: 8

As a developer, I want modal and overlay components so that I can display content on top of the main UI with proper focus management.

## Acceptance Criteria

- [ ] Modal component with backdrop
- [ ] Dialog component for confirmations
- [ ] Drawer component that slides from edge
- [ ] Sheet component for bottom sheets
- [ ] Focus trap and ESC to close
- [ ] Portal rendering and z-index stacking
- [ ] Accessibility with ARIA attributes
