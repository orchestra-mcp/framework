# OCD-26: Timer, Break & Pomodoro Reports

**Type**: Story | **Status**: backlog | **Points**: 5

As an employee, I want to view reports of my time tracking, breaks, and pomodoro sessions with export capability so that I can review my work patterns.

## Acceptance Criteria

- [ ] Time tracked per task and per project views
- [ ] Break log with duration and type
- [ ] Pomodoro session history
- [ ] Charts and summaries (daily, weekly, monthly)
- [ ] Export as CSV
- [ ] Export as PDF
- [ ] Date range filter
