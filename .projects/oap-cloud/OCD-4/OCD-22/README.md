# OCD-22: Documentation Reader & Sidebar Navigation

**Type**: Story | **Status**: backlog | **Points**: 5

As a developer, I want to browse auto-generated documentation with a navigable sidebar so that I can find information about the platform quickly.

## Acceptance Criteria

- [ ] Docs auto-read from docs/ and src/packages/*/docs/
- [ ] Sidebar navigation generated from folder structure
- [ ] README.md files serve as index pages
- [ ] Markdown rendered with same universal parser
- [ ] Breadcrumb navigation
- [ ] Edit on GitHub link per page
