# OCD-35: Laravel Echo Frontend & Real-time UI Integration

**Type**: Story | **Status**: backlog | **Points**: 5

As a frontend developer, I want Laravel Echo configured in React so that all dashboard widgets update in real time without page refreshes.

## Acceptance Criteria

- [ ] Laravel Echo + Socket.io client installed in React
- [ ] Echo configured to connect to Reverb
- [ ] Presence channels for team activity working
- [ ] Private channels for user-specific data working
- [ ] React hooks for subscribing to channels
- [ ] Real-time updates render on all dashboard widgets
- [ ] Connection status indicator in UI
