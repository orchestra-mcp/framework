# ODS-32: Form components (FormField, InputGroup, SelectMenu)

**Type**: Story | **Status**: in_progress | **Points**: 5

As a developer, I want composite form components so that I can quickly build forms with labels, inputs, and error messages in a consistent layout.

## Acceptance Criteria

- [ ] FormField component combines label, input, helper text, and error
- [ ] InputGroup component with prefix/suffix icons and buttons
- [ ] SelectMenu component with searchable dropdown
- [ ] Validation state handling
- [ ] Accessibility with proper ARIA labels
