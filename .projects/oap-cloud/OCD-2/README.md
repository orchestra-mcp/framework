# OCD-2: Authentication System

**Type**: Epic | **Status**: backlog | **Priority**: high

Complete authentication system using Jetstream + Fortify + Sanctum. Includes login, register, password reset, OTP/2FA with recovery codes, magic login links, profile management (avatar via Spatie Media Library, devices, API keys, password change), user settings via Spatie Settings, team management via Jetstream Teams, and role-based access (employee, team-owner, admin).
