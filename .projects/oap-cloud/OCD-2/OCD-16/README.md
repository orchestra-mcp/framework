# OCD-16: User Settings & Notification Preferences

**Type**: Story | **Status**: backlog | **Points**: 3

As a user, I want to manage my personal settings and notification preferences so that the platform behaves the way I prefer.

## Acceptance Criteria

- [ ] User settings page using Spatie Settings
- [ ] Settings include theme preference, timezone, language
- [ ] Notification preferences (email, in-app, push toggles)
- [ ] Settings persist and load correctly on page refresh
- [ ] Settings are scoped per user
