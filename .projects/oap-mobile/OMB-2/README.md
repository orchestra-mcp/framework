# OMB-2: Authentication Screens

**Type**: Epic | **Status**: backlog | **Priority**: high

Implement the full authentication flow mirroring the web platform: Login, Register, Reset Password, OTP/2FA input, magic login deep link handler, and biometric auth (Face ID / fingerprint). Integrate with the cloud platform's Laravel Sanctum token-based authentication. Store tokens securely using react-native-keychain. All screens use NativeWind styling and React Navigation 7 stack navigator.
