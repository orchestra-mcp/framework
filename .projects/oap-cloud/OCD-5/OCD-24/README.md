# OCD-24: Progress Widgets & Dashboard Overview

**Type**: Story | **Status**: backlog | **Points**: 5

As an employee, I want to see my progress at a glance with widgets showing tasks completed, time tracked, commits made, and current streak so that I can measure my productivity.

## Acceptance Criteria

- [ ] Dashboard page with progress widgets
- [ ] Tasks completed this week widget
- [ ] Time tracked this week widget
- [ ] Commits made this week widget
- [ ] Current streak widget
- [ ] Weekly progress chart
- [ ] Task burndown chart
- [ ] All data fetched via API and rendered in React
