# OCD-19: Support Page & Email Subscription

**Type**: Story | **Status**: backlog | **Points**: 5

As a visitor, I want to find answers to common questions, contact support, and subscribe to updates so that I can get help and stay informed.

## Acceptance Criteria

- [ ] Support page with FAQ accordion (admin-manageable)
- [ ] Contact form that sends email to admin and creates notification
- [ ] Email subscription form with double opt-in
- [ ] Unsubscribe link in emails
- [ ] FAQ CRUD in admin panel
- [ ] Subscription model with email verification
