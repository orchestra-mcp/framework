# OCD-18: Landing Page & Download Page

**Type**: Story | **Status**: backlog | **Points**: 5

As a visitor, I want to see a compelling landing page and easily download the app for my platform so that I can understand the product and get started quickly.

## Acceptance Criteria

- [ ] Landing page with hero section, feature grid, testimonials, CTA
- [ ] Download page with automatic platform detection (macOS/Windows/Linux)
- [ ] Download buttons per platform with version info
- [ ] System requirements listed
- [ ] Both pages are responsive
- [ ] Both pages use guest layout
