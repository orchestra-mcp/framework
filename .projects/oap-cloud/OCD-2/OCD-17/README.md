# OCD-17: Teams & Roles (Jetstream Teams)

**Type**: Story | **Status**: backlog | **Points**: 5

As a team owner, I want to create teams, invite members, and assign roles so that my organization can collaborate on the platform.

## Acceptance Criteria

- [ ] Team creation with name and slug
- [ ] Invite members by email
- [ ] Roles: employee, team-owner, admin with correct permissions
- [ ] Switch between teams
- [ ] Team-scoped data isolation
- [ ] Remove members from team
- [ ] Team settings page
