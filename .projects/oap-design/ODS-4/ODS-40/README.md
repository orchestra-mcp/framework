# ODS-40: Dropdown system (Dropdown, Popover, Tooltip, ContextMenu)

**Type**: Story | **Status**: backlog | **Points**: 8

As a developer, I want dropdown and popover components so that I can display contextual content and menus that are properly positioned.

## Acceptance Criteria

- [ ] Dropdown component with trigger and content
- [ ] Popover component with positioning
- [ ] Tooltip component with hover delay
- [ ] ContextMenu component for right-click menus
- [ ] Auto-positioning to stay in viewport
- [ ] Click outside to close
