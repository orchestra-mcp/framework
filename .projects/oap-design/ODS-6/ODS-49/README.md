# ODS-49: Update all existing components to use design tokens

**Type**: Story | **Status**: backlog | **Points**: 8

As a developer, I want all existing custom components updated to use design tokens so that theme changes propagate everywhere.

## Acceptance Criteria

- [ ] All hardcoded colors replaced with CSS variables
- [ ] All hardcoded spacing uses design tokens
- [ ] Typography uses design system fonts
- [ ] Shadows and borders use design tokens
- [ ] No inline color values in any component
