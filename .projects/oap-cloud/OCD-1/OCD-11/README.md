# OCD-11: Install Spatie Packages & Tailwind CSS 4 Setup

**Type**: Story | **Status**: backlog | **Points**: 3

As a developer, I want Spatie Media Library and Spatie Settings installed, and Tailwind CSS 4 configured, so that media uploads, user settings, and styling are ready for feature development.

## Acceptance Criteria

- [ ] Spatie Media Library installed and configured
- [ ] Spatie Settings installed with migration published
- [ ] Tailwind CSS 4 installed and configured with Vite
- [ ] Tailwind utility classes render correctly in React components
- [ ] Media library migration is published and runnable
- [ ] Settings migration is published and runnable
