# OCD-27: Version Control & AI Agent Control

**Type**: Story | **Status**: backlog | **Points**: 8

As an employee, I want to view my commit history, PRs, and issues, and control my AI agent remotely so that I can manage development work and AI assistance from the web dashboard.

## Acceptance Criteria

- [ ] Commit history list from tracked repos
- [ ] PR list with status (open, merged, closed)
- [ ] Issue tracking view
- [ ] AI agent status display (idle, working, waiting)
- [ ] Start task on agent button
- [ ] Approve/deny agent permission requests
- [ ] Answer agent questions
- [ ] Send new prompts to agent
- [ ] Agent output log viewer
- [ ] All agent interactions via WebSocket
