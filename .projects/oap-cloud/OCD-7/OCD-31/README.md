# OCD-31: Entity CRUD & User Management

**Type**: Story | **Status**: backlog | **Points**: 8

As an admin, I want full CRUD for all database entities and user management tools so that I can manage the entire platform.

## Acceptance Criteria

- [ ] CRUD pages for all major models (users, teams, posts, extensions, etc.)
- [ ] User management: ban, impersonate, reset password
- [ ] Team management from admin view
- [ ] Data export (CSV)
- [ ] Paginated data tables with search and filters
- [ ] Confirmation dialogs for destructive actions
