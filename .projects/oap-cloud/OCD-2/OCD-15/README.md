# OCD-15: Profile Management (Avatar, Devices, API Keys, Password)

**Type**: Story | **Status**: backlog | **Points**: 5

As a user, I want to manage my profile including avatar upload, active devices, API keys, and password changes so that I have full control over my account.

## Acceptance Criteria

- [ ] Avatar upload with crop/resize via Spatie Media Library
- [ ] Active sessions/devices list with revoke capability (Jetstream)
- [ ] API key (Sanctum token) creation with name and permissions
- [ ] API key list with delete capability
- [ ] Password change with current password verification
- [ ] Profile page is a tabbed Inertia React component
