# OCD-9: Scaffold Laravel 12 + Inertia.js + React + TypeScript

**Type**: Story | **Status**: backlog | **Points**: 5

As a developer, I want the Laravel 12 project scaffolded with Inertia.js 2, React 19, and TypeScript so that I have a working foundation to build the cloud platform on.

## Acceptance Criteria

- [ ] Laravel 12 project created at src/platform/
- [ ] Inertia.js 2 installed and configured with React 19 adapter
- [ ] TypeScript configured with strict mode
- [ ] Vite configured with React and TypeScript plugins
- [ ] Default welcome page renders via Inertia
- [ ] php artisan serve starts without errors
- [ ] npm run dev starts Vite without errors
