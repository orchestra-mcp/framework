# OCD-33: Push Updates, Marketing Notifications & Analytics

**Type**: Story | **Status**: backlog | **Points**: 8

As an admin, I want to push IDE updates, send marketing notifications, and view analytics so that I can manage releases, engage users, and track platform health.

## Acceptance Criteria

- [ ] Push new IDE version with release notes
- [ ] Send notifications to all users or targeted groups
- [ ] Target by role, team, or specific users
- [ ] Schedule notifications for future delivery
- [ ] Notification template management
- [ ] Analytics: DAU/WAU/MAU charts
- [ ] Extension install/uninstall rates
- [ ] Feature usage metrics
