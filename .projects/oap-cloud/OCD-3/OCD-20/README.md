# OCD-20: Blog System

**Type**: Story | **Status**: backlog | **Points**: 5

As a visitor, I want to read blog posts about the product so that I can stay updated on features and best practices.

## Acceptance Criteria

- [ ] Blog listing page with pagination
- [ ] Blog post detail page with markdown rendering
- [ ] Categories and tags with filtering
- [ ] Featured image per post
- [ ] SEO meta tags per post
- [ ] RSS feed at /feed
- [ ] Admin-only post creation/editing
